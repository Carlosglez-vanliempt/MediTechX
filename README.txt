  Proyecto de Universidad.

Consiste en un Sistema de Gestión de Clientes para un Centro Médico.

- Diferentes usuarios ( Médico, Paciente, Recepcionista, Administrador )
- Gestión de Citas y Atención al ciente mediante correos electrónicos
- Almacenamiento de usuarios, citas y consultas mediante ficheros JSON.
